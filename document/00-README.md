# 文件传输项目 — 设计文档索引

## 1. 文档说明

本目录存放**网关侧**文件传输功能的设计与实现说明。业务报文格式见 [04-通信协议.md](04-通信协议.md)（protocol-1.0）；代码按 TDD 实现，支持本机与 OpenWrt 交叉编译。

| 属性 | 值 |
|------|-----|
| 项目名称 | transferFile |
| 版本 | **V0.0.4**（当前） |
| 实现范围 | 目标机网关 `transferFile` + 开发机 `platform_sim` |
| 传输通道 | MQTT（libmosquitto / 内存模拟总线） |
| 开发语言 | C++17 |
| 构建系统 | CMake |
| 开发方式 | 测试驱动开发（TDD），42 项自动化测试 |

仓库根目录 [README.md](../README.md) 提供快速上手；**下一步建议**见 [13-项目状态与路线图.md](13-项目状态与路线图.md)。

## 3. 术语表

| 术语 | 说明 |
|------|------|
| 平台 | 下发召唤、接收简报与内容；联调为 `platform_sim` |
| 网关 | 本项目 `transferFile`，部署在**目标机** |
| 召唤（Summon） | MQTT 下发的文件传输请求 |
| 文件简报（Brief） | 首次应答：可否传输、CRC、大小等 |
| 文件内容（Content） | Base64 分段文件数据 |
| StartByte | 1-based 起始字节；续传起点 |
| CmdId | 命令 ID，关联一次传输 |
| FileSegNo | 单次召唤触发的内容段序号，从 1 递增 |

## 4. 快速命令

| 场景 | 命令 |
|------|------|
| 本机编译 + 测试 | `./scripts/build-native.sh` |
| **V0.0.3 验收（42 项）** | `./scripts/run-acceptance-tests.sh` |
| 平台推送联调 | `build/platform_sim ... --push-file <本地> --gateway-path <目标机路径>` |
| 交叉编译网关 | `./scripts/build-openwrt.sh` |
| **打包目标机部署** | `./scripts/package-target.sh` |
| 联调说明 | `./scripts/debug-mqtt-local.sh` |
| 目标机网关 | `build-openwrt/transferFile -c config/transferFile.gateway.target.json` |
| 开发机平台 | `build/platform_sim -c config/transferFile.platform.json --gateway-file <路径>` |
