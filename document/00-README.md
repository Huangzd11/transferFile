## 3. 术语表

| 术语 | 说明 |
|------|------|
| 平台 | 下发召唤指令、接收简报与文件内容的一方（生产环境为既有系统；联调可用 `platform_sim`） |
| 网关 | 本项目 `transferFile`；订阅 MQTT 召唤，应答简报与内容 |
| 召唤（Summon） | 平台通过 MQTT 下发的「请求传输某文件」指令 |
| 文件简报（Brief） | 网关对召唤的首次应答，含是否可传、文件元信息等 |
| 文件内容（Content） | 网关在一次会话中按块发送的实际文件字节 |
| 会话（Session） | 一次从召唤到传完或中止的逻辑过程，由 **CmdId** 关联 |
| StartByte | 协议中的起始字节位置，**从 1 开始**；网关内部转换为 0-based 偏移 |
| CmdId | 命令 ID，标识一次文件请求/传输（0～4294967295） |
| FileSegNo | 单次召唤触发的内容发送序列中的段序号，从 1 递增 |
| 内容块（Chunk） | 单次内容帧承载的文件数据片段 |
| 有效交互 | 用于重置超时计时器的行为：新的合法召唤（见 04、03 章） |
| Broker | MQTT 消息代理服务器 |

## 4. 快速命令

| 场景 | 命令 |
|------|------|
| 本机编译 + 测试 | `./scripts/build-native.sh` |
| 交叉编译网关 | `./scripts/build-openwrt.sh` |
| 联调说明 | `./scripts/debug-mqtt-local.sh` |
| 网关运行 | `./build/transferFile -c config/transferFile.json` |
| 平台模拟 | `./build/platform_sim -c config/transferFile.platform.json --demo` |
