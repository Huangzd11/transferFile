## 3. 目标机手工验收

### 3.1 召唤 + 内容确认

**目标机**：

```bash
./transferFile -c transferFile.gateway.target.json
```

**开发机**：

```bash
./build/platform_sim -c config/transferFile.platform.json \
  --gateway-file /tmp/v004_test.bin
```

**预期**：

1. 网关日志：`<<< 收到文件内容确认`（每段一条）
2. 平台日志：`<<< 已发布内容确认`
3. 平台 `联调成功`，文件内容与目标机源文件一致

### 3.2 超时 spot check

发召唤后平台**故意不回**内容确认，等待 `timeoutSec` → 网关 `召唤上传中止` / 超时。
